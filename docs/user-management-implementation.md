## 技術仕様

### 使用技術
- **フロントエンド**: Next.js 14 (App Router)
- **UIライブラリ**: TanStack Table + Shadcn/ui
- **状態管理**: TanStack Query + Redux Toolkit
- **バックエンド**: Laravel API
- **データベース**: PostgreSQL

### ファイル構成
```
frontend/src/
├── lib/
│   ├── userService.ts          # ユーザーAPIサービス
│   └── types.ts               # 共通型定義
├── hooks/
│   └── useUsers.ts            # ユーザー管理フック
├── components/
│   ├── ui/                    # Shadcn/uiコンポーネント
│   ├── common/
│   │   └── data-display/
│   │       └── DataTable/     # 汎用テーブル
│   └── features/
│       └── users/             # ユーザー管理機能
└── app/
    └── (management)/
        └── users/
            └── page.tsx       # ユーザー一覧ページ
```

### API仕様
- **エンドポイント**: `/api/users`
- **認証**: Laravel Sanctum
- **レスポンス形式**: JSON
- **ページネーション**: サーバーサイド
